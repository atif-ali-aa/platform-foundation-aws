# assets/

Static images referenced from documentation (README badges, exported
diagram images) that aren't better expressed as inline Mermaid.

Architecture diagrams themselves live in [`architecture/`](../architecture)
as Mermaid source so they render directly on GitHub and stay diffable.
This directory is for anything Mermaid can't express well (e.g. exported
PNGs for platforms that don't render Mermaid).
